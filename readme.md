# 自动玩微信小游戏跳一跳

### Requirement

- Python
- Opencv
- Adb tools
- Android Phone

### Run

	python play.py 

Here is a video demo.

### If you fail

I just collect few samples for matching. If you happened to fail, you can turn on the debug mode,
tailor the jump target as shown in the `source/circle` or `source/table` dir.
